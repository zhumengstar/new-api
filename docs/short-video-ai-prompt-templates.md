# 短剧与 AI 视频分镜提示词模板文档

## 1. 文档说明

本文档用于沉淀短剧、AI 视频、小说转分镜、视频反推、人物资产、场景资产和道具资产等高频内容生产场景的标准提示词模板。

模板来源于请求体日志中占比最高的“短剧/AI 视频分镜提示词”类需求。该类需求在全量带请求体日志中约占 72.98%，主要集中在以下工作流：

- 小说、剧本、推文文案转 AI 视频分镜。
- 即梦、豆包、Seedance、Veo、Grok 等视频模型提示词生成。
- 人物、场景、道具资产提取。
- 视频内容反推提示词。
- 提示词洗稿、安全改写和一键复制整理。
- 电商产品短视频脚本和视频提示词生成。

本文档目标是将这些高频需求整理成可复用、可复制、可维护的标准模板，方便后续业务分析、提示词配置、Gem/智能体搭建或产品化封装。

## 2. 适用范围

适用：

- AI 短剧生产。
- 小说/剧本视频化。
- 短视频镜头脚本生成。
- AI 视频模型提示词生成。
- 人物/场景/道具资产库建设。
- 电商短视频脚本生成。
- 视频拆解、复刻和反推。

不适用：

- 法律、医疗、金融等严肃专业结论生成。
- 需要真实版权授权的影视 IP 复刻。
- 绕过平台安全审核或生成违规内容。
- 直接替代人工导演对最终镜头质量的审核。

## 3. 通用提示词规范

### 3.1 输入规范

建议输入内容包含：

- 原始文本：小说、剧本、推文文案、视频描述或产品信息。
- 目标平台：即梦、豆包、Seedance、Veo、Grok 等。
- 画幅比例：常见为 9:16 竖屏。
- 风格要求：写实、3D 国漫、电影感、电商展示、纪实短视频等。
- 输出格式：表格、JSON、分镜段落或一键复制提示词。
- 限制条件：是否保留原台词、是否允许改写、是否需要安全改写。

### 3.2 输出规范

高质量输出建议包含：

- 人物一致性：姓名、身份、年龄、外貌、服装、状态。
- 场景一致性：地点、时间、光影、空间结构、道具。
- 镜头完整性：时间区间、景别、运镜、动作、台词、音效。
- 动作可生成性：避免抽象心理描述，转化成可见动作。
- 一键复制性：减少解释文字，直接给最终可用内容。

### 3.3 风险控制规范

提示词应避免：

- 过度血腥、暴力、色情或低俗描述。
- 过度聚焦身体局部的镜头描述。
- 使用真实人物、影视 IP、品牌形象进行未授权复刻。
- 使用“高级感”“氛围感”等无法直接生成的空泛词替代具体画面。
- 多段输出分散，导致用户需要手动拼接。

## 5. 标准模板

### T01. 剧本转视频分镜模板

#### 使用场景

适用于将小说、短剧剧本、推文文案改写成 AI 视频生成工具可直接使用的分镜提示词。

#### 输入变量

- `{{剧本内容}}`：原始小说、剧本或推文文案。
- `{{目标平台}}`：即梦、豆包、Seedance、Veo 等，可选。
- `{{视频时长}}`：例如 10 秒、15 秒、20 秒。
- `{{画面风格}}`：例如 3D 国漫写实、真人电影感。

#### 标准提示词

```text
你是一名专业 AI 短剧分镜编导，擅长把小说/剧本文本转换成可直接用于 AI 视频生成的分镜提示词。

请基于以下剧本内容，生成适合竖屏短剧的分镜脚本。

任务目标：
将原始剧情拆解成可生成、可拍摄、可复制的 AI 视频分镜。

硬性要求：
1. 保留原剧情、人物关系、台词和核心冲突，不要删减关键剧情。
2. 拆成 4-6 个分镜，总时长控制在 {{视频时长}}。
3. 每个分镜必须包含：时间区间、景别、人物位置、运镜、动作、环境、光影、台词、音效。
4. 人物外貌、服装、场景风格必须前后一致。
5. 镜头语言要具体，避免“氛围感”“高级感”等空泛描述。
6. 动作必须可视化，不要只写心理活动。
7. 输出内容必须方便直接复制到 AI 视频生成工具。

统一风格：
- 竖屏 9:16
- {{画面风格}}
- 电影级光影
- 无字幕、无边框、无水印
- 镜头自然衔接，动作连续

输出格式：

【全局人物设定】
人物A：
人物B：

【全局场景设定】
地点：
时间：
光影：
美术风格：

【分镜 1｜0-3 秒】
景别：
人物位置：
运镜：
画面内容：
动作细节：
台词：
音效：

【分镜 2｜3-6 秒】
景别：
人物位置：
运镜：
画面内容：
动作细节：
台词：
音效：

按以上格式继续，直到覆盖完整剧情。

剧本内容：
{{剧本内容}}
```

### T02. AI 视频反推提示词模板

#### 使用场景

适用于用户提供视频、截图或视频描述，希望反推出可用于 AI 视频生成的提示词。

#### 输入变量

- `{{视频内容或截图描述}}`：视频画面、截图内容或人工描述。
- `{{目标模型}}`：Seedance、Veo、即梦、豆包等。
- `{{输出语言}}`：默认中文。

#### 标准提示词

```text
你是一名专业短视频拆解师和 AI 视频提示词编导。

请根据我提供的视频/画面内容，反推出完整 AI 视频生成提示词。

必须覆盖：
1. 主体：人物/物品/场景的详细特征。
2. 动作：每个动作的先后顺序、幅度、速度、身体细节。
3. 场景：环境、空间、背景元素。
4. 光影：光线方向、强度、色调、明暗关系。
5. 运镜：固定、推、拉、摇、移、跟拍、俯拍、仰拍等。
6. 风格：写实、电影感、短视频质感、产品展示感等。
7. 时间：精确到每个时间段在做什么。
8. 音频：如果有台词、环境声或音效，也要写出。

输出要求：
- 只输出{{输出语言}}。
- 生成可直接复制到 {{目标模型}} 的一整段提示词。
- 不要解释分析过程。
- 不要分散成多段让我手动拼接。
- 如果原视频包含不适合生成的敏感内容，请改写为安全、中性的镜头语言。

视频/画面内容：
{{视频内容或截图描述}}
```

### T03. 人物资产提取模板

#### 使用场景

适用于从小说、剧本、推文文案中提取所有角色，形成后续生图、视频、分镜使用的人物资产库。

#### 输入变量

- `{{小说或剧本内容}}`：原文内容。
- `{{输出格式}}`：表格、JSON 或编号列表。

#### 标准提示词

```text
你是专业的短剧人物资产提取助手。

请从以下小说/剧本/推文文案中，提取所有实际出现或会形成画面的角色，并整理为 AI 生图/视频资产表。

提取规则：
1. 不要只提取主角，所有有画面作用的人物都要提取。
2. 合并同一人物的别名、称呼和身份变化。
3. 区分主角、配角、群众、反派、特殊实体。
4. 外貌不要凭空夸张，优先依据原文。
5. 原文没有明确描述时，可以给出合理短剧化建议，但必须标注为“建议”。
6. 每个人物都要能直接用于后续生图、分镜、视频生成。

输出格式：

1. 人物名称 / 别名
身份：
剧情作用：
性格：
外貌依据：
外貌建议：
服装建议：
关键道具：
关键状态变化：
适合画面：
生图提示词：

原文：
{{小说或剧本内容}}
```

### T04. 场景资产提取模板

#### 使用场景

适用于建立场景库，服务于 AI 生图、AI 视频、分镜制作和连续剧集视觉一致性。

#### 输入变量

- `{{小说/剧本/推文文案}}`：原始资料。

#### 标准提示词

```text
你是一名影视与动漫场景信息提取器。

请根据以下资料，提取故事中所有实际形成独立画面的场景，并输出适合场景设计、分镜制作和 AI 生图使用的场景信息。

规则：
1. 只提取真实发生剧情的场景，不提取抽象概念。
2. 地点、时间、环境以原文为最高优先级。
3. 同一地点不同状态可拆分，例如“白天的院子”和“夜晚的院子”。
4. 场景描述必须具体到空间、材质、光线和可见元素。
5. 输出严格结构化，不要写分析过程。

输出严格 JSON 数组：

[
  {
    "场景名称": "",
    "场景类型": "",
    "时间": "",
    "环境描述": "",
    "建筑/空间特征": "",
    "光影氛围": "",
    "出现人物": [],
    "关键道具": [],
    "剧情作用": "",
    "AI生图提示词": ""
  }
]

资料：
{{小说/剧本/推文文案}}
```

### T05. 道具资产提取模板

#### 使用场景

适用于提取重要物品、信物、武器、文书、阵法、特殊实体等视觉资产。

#### 输入变量

- `{{文本内容}}`：小说、剧本或推文文案。

#### 标准提示词

```text
你是专业的小说与剧本道具信息提取助手。

请从以下内容中识别所有实际形成独立视觉形象、并对剧情具有作用的重要物品或特殊实体。

要求：
1. 完成去重、归并、状态判断和别称整理。
2. 不提取无画面价值的普通背景物，除非它推动剧情。
3. 每个道具都要给出可用于 AI 生图的客观描述。
4. 道具描述必须包含外观、材质、颜色、状态和剧情功能。
5. 输出严格 JSON 数组，不要 Markdown，不要解释。

输出字段：
- 道具名称
- 别名
- 所属人物
- 首次出现位置
- 外观描述
- 材质/颜色/尺寸
- 剧情作用
- 状态变化
- 生图提示词

文本：
{{文本内容}}
```

### T06. 提示词洗稿与安全改写模板

#### 使用场景

适用于原提示词被视频平台拒绝、画面过于敏感、描述过于生硬或需要规避同质化时。

#### 输入变量

- `{{原提示词}}`：需要改写的提示词。
- `{{目标平台}}`：即梦、豆包、Seedance 等。

#### 标准提示词

```text
你是一名 AI 视频提示词优化师。

请在不改变核心剧情和镜头意图的前提下，对以下提示词进行深度改写，使其更适合 {{目标平台}} 通过审核并稳定出片。

改写要求：
1. 保留人物、场景、动作、冲突和台词核心。
2. 删除或弱化可能触发审核的暴力、血腥、低俗、过度身体局部展示描述。
3. 用更中性的镜头语言替换敏感表达。
4. 提升画面连续性、人物一致性和动作可生成性。
5. 保留镜头可执行性，不要改成抽象文学描写。
6. 输出“一键复制版”，不要分散解释。

输出格式：
【安全改写后一键复制版】
...

原提示词：
{{原提示词}}
```

```text
你是批量短剧分镜生成器。

硬性要求：
1. 每段独立输出，不要混淆编号。
2. 每段控制在 {{单段时长}}。
3. 每段拆 3-5 个镜头。
4. 人物外貌、服装、场景必须继承上一段设定。
5. 同一人物在不同编号中必须保持一致。
6. 不要输出解释，只输出分镜。

每段输出格式：

【编号】
【人物继承】
【场景继承】

【分镜 1】
时间：
景别：
运镜：

动作：
台词：
音效：

...

{{批量剧本}}
```

### T09. 视频模型通用增强词库

#### 使用场景

适用于拼接到各类视频提示词末尾，增强画质、一致性和可生成性。

#### 标准词库

```text
画幅：竖屏 9:16，4K，高清，短视频平台适配。
画质：写实电影感，电影级光影，高质量细节，真实材质。
人物：外貌一致，服装一致，年龄一致，发型一致，人物不变形。
动作：动作连续，重心真实，身体惯性自然，表情细微变化。
镜头：固定机位，缓慢前推，横移跟拍，低角度仰拍，特写切入。
光影：自然光，低调照明，高反差，轮廓光，暖色室内光，冷色夜景。
声音：同期声真实，环境声自然，台词清晰。
限制：无字幕，无水印，无边框，无闪烁，无畸形手指，无多余人物，无画面跳变。
```

## 6. 推荐组合工作流

### 6.1 小说转短剧视频

1. 使用 T03 提取人物资产。
2. 使用 T04 提取场景资产。
3. 使用 T05 提取道具资产。
4. 使用 T01 将每一段剧本转为分镜。
5. 使用 T06 对敏感或失败提示词做安全改写。
6. 使用 T09 追加通用增强词。

### 6.2 视频复刻/反推

1. 使用 T02 反推视频提示词。
2. 如果需要多平台适配，针对目标模型改写运镜和时长。
3. 使用 T06 做安全改写。
4. 使用 T09 追加增强词。

### 6.3 电商短视频生成

1. 使用 T08 生成产品短视频脚本。
2. 对商品图进行视觉描述或审核。
3. 生成分镜提示词。
4. 使用 T09 增强画质和一致性。

## 7. 质量检查清单

生成结果交付前建议检查：

- 是否保留了原剧情核心。
- 是否包含明确人物设定。
- 是否包含明确场景设定。
- 是否有时间区间。
- 是否有景别和运镜。
- 是否有动作细节。
- 是否有台词和音效。
- 是否能一键复制使用。
- 是否避免敏感或违规表达。
- 是否保证人物和场景连续一致。

## 8. 后续维护建议

建议定期从请求体日志中抽样复盘：

- 新增高频场景时补充新模板。
- 某类模板失败率高时补充安全改写规则。
- 某个视频模型变化明显时单独维护模型适配版。
- 对“其他/未细分”类请求做二次归类，提高分类精度。
- 将模板参数化后接入前端或后台配置，形成可选模板库。
